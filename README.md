# Sistema de Gestión de Flota Uber
Este programa fue desarrollado en **Python** para controlar el mantenimiento de una flotilla de transporte en CDMX.

### Funciones:
- Registro de modelo, chofer y kilometraje.
- Alerta automática cuando el vehículo requiere taller (5,000 km).
- Guardado de registros en archivo de texto (.txt).
